# nodemongo-crud

Simple Node.js + MongoDB CRUD example project demonstrating a minimal REST API for managing a collection (items).
Project details: https://www.coursera.org/learn/showcase-build-a-crud-nodejs-and-mongodb-employee-management-web-app/

Part of the code / doc is generated by AI (copilot).


## Features
- Create, read, update, delete (CRUD) operations
- Express-based REST API
- MongoDB (mongoose) for persistence
- Environment-driven configuration

## Prerequisites
- Node.js 18+ (or compatible)
- npm or yarn
- MongoDB (local, Atlas, or Docker)

## Installation
1. Clone the repo:
    git clone <repo-url>
2. Install dependencies:
    npm install

## Configuration
Create a `.env` file at project root with:
```
PORT=3000
MONGO_URI=mongodb://localhost:27017/nodemongo
```

## Run
Start the server:
```
npm start
```
Development with hot reload:
```
npm run dev
```

## API Endpoints
Base URL: `http://localhost:3000`

- GET /api/users
  - Returns list of users
- GET /api/users/:id
  - Returns a single user by id
- POST /api/users
  - Create a new item
  - Body (JSON):
     {
        "name": "Sample",
        "password": "Required",
        "email": "Optional
     }
- PUT /api/users/:id
  - Update an user by id
  - Body: partial or full user object
- DELETE /api/users/:id
  - Delete an user by id

Example curl (create):
```
curl -X POST http://localhost:3000/api/users \
  -H "Content-Type: application/json" \
  -d '{"name":"Test User","password":"desc"}'
```
